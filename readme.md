# Live Stock Market Dashboard

A real-time stock market dashboard built using Python, Streamlit, Yahoo Finance, Plotly, and Pandas.

## Features

* Live stock price tracking
* Daily return calculation
* Trading volume analysis
* Volatility measurement
* Buy/Sell signal using 20-Day Moving Average
* Interactive price trend chart
* Candlestick chart visualization
* Multi-stock comparison
* Top Gainer and Top Loser analysis
* Custom watchlist
* Stock rankings
* Market news feed
* Technical stock screener

## Technologies Used

* Python
* Streamlit
* Yahoo Finance (yfinance)
* Plotly
* Pandas
* Feedparser
* TA (Technical Analysis Library)

## Installation

1. Clone or download the repository.

2. Install required libraries:

pip install -r requirements.txt

3. Run the application:

python -m streamlit run app.py

4. Open the local URL generated by Streamlit in your browser.
